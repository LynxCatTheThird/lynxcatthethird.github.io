条款与政策

协议最新更新时间为：2025 年 2 月 19 日

当您访问本站时，默认您已知晓并允许本站收集您的信息，并明确且同意以下条款。若您不同意本协议的任何条款，请停止访问并关闭本站。

隐私政策

个人信息收集

在访问时

本站会收集以下信息：

  • 访问次数和访问信息（包括但不限于您的 IP 地址、运营商信息、国家与地区、访问来源、用户代理等信息）
  • 设备信息（如操作系统、浏览器版本等）
  • 访问来源（例如，从搜索引擎、社交媒体等渠道访问本站）
  • 访问过程（包括但不限于浏览时长、操作方式、浏览页面、性能与网络加载情况等）
  • 以上信息可能涉及通过第三方服务（如 Github Pages、Vercel、Netlify、Cloudflare 等）收集，具体详情请参考相关服务的隐私协议。

获取与使用方式

  • 第三方服务商（如 Github Pages、Vercel、Netlify、Cloudflare、MongoDB 等）收集和记录您的访问信息。本站无法确保这些第三方服务如何使用您的信息，请仔细查阅其隐私政策。

使用目的

本站会基于以下目的使用您的个人信息：

  1. 优化文章内容和网站性能；
  2. 监控和优化网站加载情况；
  3. 分析访问和点击数据，以改善用户体验；
  4. 反恶意行为监测（包括针对网站的恶意访问和言论）；
  5. 展示浏览数据（例如访问量统计）；
  6. 搜索结果优化。

在评论时

收集信息

当您使用评论功能时，本站将收集以下额外信息：

  • 邮箱、QQ 号（如果提供）、昵称、头像
  • IP 地址、浏览器用户代理、IP 归属地等信息。

获取与使用方式

  • 使用 Twikoo 服务时，站点会收集您的昵称、邮箱、IP 归属地（精确到省级）、用户代理等信息。
  • 如果您提供了 QQ 号，系统会根据您的 QQ 邮箱或 QQ 号关联头像。

使用目的

本站将用于以下目的：

  1. 网站和文章的优化；
  2. 监控访问情况，提高网站性能；
  3. 反恶意行为（包括针对评论的恶意内容）；
  4. 在评论中展示您头像（如有提供）；
  5. 评论时记录您的邮箱用于通知回复；
  6. IP 地址和浏览器信息用于反垃圾评论、定位问题等。

Cookies 与本地存储

  • 本站使用 Cookies 和 LocalStorage 来记忆用户偏好（如评论时填入的信息、深浅色模式选择等）以及统计信息（如访问量、用户行为等）。
  • 您可以随时清除浏览器中的 Cookies 和 LocalStorage 信息，这不会影响您正常使用本站。

用户权利

在我力所能及的范围内，您有权：

  • 请求访问您的个人数据；
  • 请求删除您的个人数据；
  • 撤回同意，我们将停止处理您的个人信息。

数据安全

我们采取必要的技术措施（如加密、访问控制等）保护您的个人信息免遭未经授权的访问、披露、修改或毁坏。然而，由于网络环境的开放性，无法绝对保证数据传输的安全。

适用法律与争议解决

本隐私政策适用于全球范围内的用户，但针对不同地区的用户，适用的法律法规可能有所不同。若有争议，将根据相关法律进行处理。
